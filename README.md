# mc-server-spigot-1.16.2

Minecraft Mod / Plugin 開発用テンプレートコンテナ

## コマンドリファレンス

### コンテナ起動

`docker-compose up -d`

### コンテナにアタッチ

`docker exec -it mc-server-spigot-1.16.2 /bin/bash`


## jarファイルパス

`/opt/minecraft/~`
